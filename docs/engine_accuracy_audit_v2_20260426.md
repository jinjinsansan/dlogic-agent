## 6. 推奨運用ルール (3層構造)

### Tier 1: 最優先 (穴党参謀チャンネル既設)
- **NAR平日厳格パターン × 単勝** (454.6%)
- 火水木の強5会場 + 6-12頭 + 5-8人気 + 2-3エンジン一致

### Tier 2: 新規追加候補 (高再現性)
- **dlogic JRA TOP3 BOX ワイド 3点** (125.6%)
  - JRA全レース対象、dlogic top3 を BOX 3点で買う
  - 1日10レース対象なら投資3,000円、期待回収3,768円 (+25%)

### Tier 3: 戦略的補助 (専門家向け)
- **2/4一致 × 9+人気 軸ワイド流し** (142.5%)
  - 大穴狙い、サンプル少なめだが特異点として記録

  - 堅実、人気馬寄り
- **4/4一致 × 6-8人気 複勝** (243.9%)
  - 出現頻度低い (週1回程度)

## 8. データ制約・注意点 (誠実に)

### サンプル制約
- 期間6週間 (2026-03-11〜04-25) のみ
- セグメント切ると30〜200件規模が多い
- 再現性は3〜6ヶ月でクロス検証推奨

### 払戻データ制約
- 1,826件マッチ (84.4%) — 残り16%はPCKEIBAデータが部分欠落
- 主に「特定日のJRA未着」「南関東の一部レース」
- 厳格パターン分析にはほぼ影響なし (NAR強5会場は揃っている)

### 戦略間の相関
- 複数戦略を同時運用すると、同じレースで複数買い目になり投資が膨らむ
- 1日の合計投資額をモニタする必要あり

### "100%超え戦略" の落とし穴
- サンプル100-300件で +10-25% は統計的にはまだギリギリ
- 1ヶ月続けて期待値どおりの推移を必ず確認すること

## 10. v1 → v2 で確実になった事実

1. **エンジン群は単勝精度では平凡 (75〜80%)** だが、**複勝・ワイドではJRAで黒字化**
2. **dlogic と metalogic は JRA向けエンジン**として強い
3. **viewlogic は単勝弱いがJRAワイドでは伏兵**
4. **2/4一致** は単勝・ワイド・複勝すべてで重要な信号
5. **4/4一致 × 中穴 複勝** という単勝とは真逆の特異点が存在
