## 1. 重要な前提・制約

### JRA データはリーク汚染で無効

- **R2 にナレッジファイルの過去履歴が無い**ため (2025/9 + 2026/3-4のみ)、過去レースを当時のナレッジで予想する技術的手段がない
- 第1セッション (`--race-type all`) で取得した JRA分は **現ナレッジ (2026/4/23時点で過去結果込み) で過去予想** = エンジンが正解を記憶している状態
- 結果: dlogic JRA 単勝率 **22% → 38%**、ワイドBOX 回収率 **126% → 282%** という非現実的上振れ
- v3 は **NAR limited で全分析を再構築** している

### NAR の軽微なリーク
- NAR分のナレッジ更新が JRAより質量・頻度ともに低いため、リーク影響は限定的と推定
- 実際、NAR単体の数字 (dlogic NAR 70.8%, metalogic 73.9%) は v2 と一致 → 信頼可能

### サンプル数の現実
- engine_hit_rates の NAR レコード: **26,808 行** (拡張後)
- enriched races (4エンジン揃 + odds + 結果あり): **1,309 races** (NAR limited)
- 第1セッション (60日分) のみで進行、残り5セッションはバックグラウンド進行中
