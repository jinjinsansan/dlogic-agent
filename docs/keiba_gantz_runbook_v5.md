## 4. 期待値と注意事項

### バンクロール推奨
- 信頼度・最高のみ → ¥10,000 (1年で +¥56k)
- 信頼度・高のみ → ¥30,000 (1年で +¥106k、連敗10日対応)
- 両方 → ¥50,000 (連敗時の余裕あり)

### 連敗時の心構え
- 信頼度・最高 = 17日連敗もあり得る (4日 = ¥1,200損失)
- 信頼度・高 = 10日連敗、DD ¥12,580
- 「3-4連敗後に大爆発」のミーンリバージョンを覚えておく (回収率218-372%)

### 季節性
- 4-5月、9-12月 = 苦戦傾向
- 7-8月、2-4月 = 好調
- 半年単位での評価が必要、月単位の収支変動は織込み済み
